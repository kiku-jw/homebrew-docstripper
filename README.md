# Homebrew Tap for DocStripper

Homebrew tap for [DocStripper](https://github.com/kiku-jw/DocStripper) - AI-powered batch document cleaner.

## Installation

```bash
brew tap kiku-jw/docstripper
brew install docstripper
```

## Usage

```bash
docstripper document.txt
docstripper file1.txt file2.docx file3.pdf
```
